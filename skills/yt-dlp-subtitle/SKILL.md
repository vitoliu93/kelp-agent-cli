# yt-dlp Subtitle Extractor

Advanced subtitle extraction from YouTube and 1000+ supported video platforms using yt-dlp.

## Features

- **Multiple languages** - Download subtitles in multiple languages (e.g., en,es,fr)
- **Multiple formats** - VTT, SRT, JSON3, ASS, SUB
- **Auto and manual subtitles** - Auto-generated by default, option to download all available
- **Fixed output path** - Unified `/tmp/yt-dlp-subtitles/` directory
- **Batch-friendly** - Success/failure feedback and file listing
- **Error handling** - Validates URLs and formats, provides helpful error messages

## Requirements

- yt-dlp installed (user confirms this)
- Valid video URL (YouTube, Vimeo, Bilibili, TED Talks, etc.)

## Quick Start

```bash
./yt-dlp-subtitle.sh "https://www.youtube.com/watch?v=VIDEO_ID"
```

Downloads English auto-generated subtitles in VTT format to `/tmp/yt-dlp-subtitles/`.

## Usage Examples

**Single language, default format (VTT):**
```bash
./yt-dlp-subtitle.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ"
```
Output: `/tmp/yt-dlp-subtitles/video-title.en.vtt`

**Multiple languages in SRT format:**
```bash
./yt-dlp-subtitle.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ" -l en,es,fr -f srt
```

**All available subtitles (not just auto-generated) to custom directory:**
```bash
./yt-dlp-subtitle.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ" -a -o ./my_subs
```

**All available languages in ASS format:**
```bash
./yt-dlp-subtitle.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ" -l all -f ass
```

## Options

| Flag | Argument | Default | Description |
|------|----------|---------|-------------|
| `-l, --lang` | code(s) | `en` | Comma-separated language codes (e.g., en,es,fr) or `all` for all available |
| `-f, --format` | format | `vtt` | Output format: vtt, srt, json3, ass, sub |
| `-o, --output` | path | `/tmp/yt-dlp-subtitles` | Output directory for subtitle files |
| `-a, --all-subs` | - | - | Download all available subtitles (not just auto-generated) |
| `-h, --help` | - | - | Display help message |

## Subtitle Formats

- **VTT** (WebVTT) - Web format, widely supported
- **SRT** (SubRip) - Common subtitle format, compatible with most players
- **JSON3** - JSON format for programmatic use
- **ASS** (Advanced SubStation Alpha) - Rich format with styling
- **SUB** (MicroDVD) - Legacy format

## Output

Files are saved to `/tmp/yt-dlp-subtitles/` with naming pattern: `<video-title>.<language>.<format>`

Example output:
```
/tmp/yt-dlp-subtitles/video-title.en.vtt
/tmp/yt-dlp-subtitles/video-title.es.vtt
/tmp/yt-dlp-subtitles/video-title.fr.vtt
```

File listing is automatically displayed after download.

## Error Handling

- **Invalid URL**: Script validates and provides feedback
- **No subtitles available**: Indicates video may not support requested subtitles
- **Format validation**: Confirms format before processing
- **Missing yt-dlp**: Clear error message if yt-dlp is not installed
